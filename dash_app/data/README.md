# Dash 看板数据说明

本目录只保留数据说明，不上传真实数据文件。

运行 `dash_app/main.py` 前，请在本目录下自行放入以下文件：

| 文件 | 说明 |
|---|---|
| `era5历史数据.xlsx` | 省级月度 ERA5 指标主数据 |
| `ONI.xlsx` | ONI 指数，用于 ENSO 分析 |
| `geo_plotly.json` | 中国省级地图底图 |

## `era5历史数据.xlsx` 最小字段

Dash 看板默认读取以下字段：

| 字段 | 说明 |
|---|---|
| `年` | 年份 |
| `月` | 月份 |
| `province` | 省份或区域名称 |
| `温度_平均_2米_省内` | 2 米平均气温 |
| `温度_最大_2米_省内` | 2 米最高气温 |
| `温度_最小_2米_省内` | 2 米最低气温 |
| `光辐照_平均_地表_省内` | 平均地表向下太阳辐射 |
| `光辐照_最大_地表_省内` | 最大地表向下太阳辐射 |
| `风速_平均_10米_省内` | 10 米平均风速 |
| `风速_最大_10米_省内` | 10 米最大风速 |
| `风速_平均_100米_省内` | 100 米平均风速 |
| `风速_最大_100米_省内` | 100 米最大风速 |
| `地表总降水_平均_省内` | 平均降水 |
| `地表总降水_最大_省内` | 最大降水 |

如果你的列名不同，需要同步调整 `main.py` 中的变量映射。

## `ONI.xlsx` 最小字段

| 字段 | 说明 |
|---|---|
| `YR` 或 `年` | 年份 |
| `MON` 或 `月` | 月份 |
| `ANOM` 或 `ONI` | ONI 指数 |

## `geo_plotly.json`

用于中国省级地图绘制。省份名称需要能与主数据中的 `province` 字段对应。
